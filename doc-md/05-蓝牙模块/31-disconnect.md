## .disconnect(delay)
关闭链接 **注意小米协议的蓝牙设备，退出插件的时候，一定要调用此方法，关闭蓝牙连接，否则下次打开插件的时候，会提示蓝牙无法连接**

**Kind**: instance function  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| delay | <code>int</code> | <code>0</code> | 延迟时长(毫秒) |
