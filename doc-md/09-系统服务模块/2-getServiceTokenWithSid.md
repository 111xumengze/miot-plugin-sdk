## .getServiceTokenWithSid(sid) ⇒ <code>Promise</code>
传入域名返回 serverToken 等信息，目前只支持小爱音箱的域名
android暂时不支持此方法

**Kind**: static function  

| Param | Type | Description |
| --- | --- | --- |
| sid | <code>string</code> | 域名，类似"xxx.xiaomi.com" |
