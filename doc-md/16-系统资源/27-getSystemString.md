## .getSystemString(key, ...params) ⇒ <code>string</code>
根据主键名获取系统的国际化字符串

**Kind**: static function  

| Param | Type | Description |
| --- | --- | --- |
| key | <code>string</code> | 主键名 |
| ...params | <code>any</code> | 参数 |

**Example**  
```js
res.getSystemString('mijia')
```
