## .openOneTimePassword(did, interval, digits)
打开一次性密码设置页

**Kind**: static function  

| Param | Type | Description |
| --- | --- | --- |
| did | <code>\*</code> | 设备did |
| interval | <code>\*</code> | 时间间隔，即密码组的刷新时间间隔，单位为分钟，类型为 number，传入 10 到 60 的整数 |
| digits | <code>\*</code> | 密码位数，类型为 number，传入 6 到 8 的整数 |
