## miot/host/storage
本地轻量级存储服务, 键值对格式
对于key-value的数据存储，我们提供的能力具体如下：
1、单个key-value的数据存储  2、批量key-value的数据存储
注意事项：
1、目前删除设备没有做自动清除本地缓存的能力(计划安排中)

**Export**: public  
**Doc_name**: 本地存储模块  
**Doc_index**: 6  
**Doc_directory**: host  
**Example**  
```js
import { Host} from "miot";
Host.storage.get("key-1").then(res => {
  console.log("res", res)
}).catch(error => {
   console.log("error", error)
});
```
